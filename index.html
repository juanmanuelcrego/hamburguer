Al final de la explicación está el botón para acceder a la página

Lorena, buenas tardes, paso a explicarte paso por paso lo implementado:

En primer lugar se implementa bootstrap para utilizar el grid system, para hacerlo se pega en el META de cada html el link hacia las hojas de estilos de bootstrap. Posteriormente se pegan también los scripts de bootstrap en cada html, antes de cerrar el body.

Se implementa también Font Awesome para utilizar iconos en el header y en algunos botones, más específicamente iconos de carro y de menu.

Se implementa Google Fonts para utilizar una fuente diferente a las que están por defecto en el sistema operativo, y para no utilizar la Roboto de bootstrap

Estructura del HTML:

  • 01- home.html
  • BOOTSTRAP
  • .container que contiene todos los elementos hijos para hacer el grid system
  • .col .header para utilizar el sistema de columna de bootstrap, además de especificar que es un header para trabajarlo en mi hoja de estilos
  • HEADER
  • h5 con título de la web
  • section class .cart-menu con dos iconos de Font Awesome uno de carro de compras y otro de menu
  • Dentro del menu dos links mediante elementos de tipo 'a' hacia otros html
  • CAROUSEL
  • previo a la implementacion del carousel de bootstrap se crea un div con la clase .row para especificar que está por fuera del header
  • CARDS
  • Se crea un nuevo row y se hacen cuatro elementos .card de bootstrap con un div para cada uno.
  • FOOTER
  • Se declara un div con clase .row para contener un div de clase .col .footer donde coloqué un p con un link a mi página de instagram
  • 02- about.html
  • Comparte los mismos parámetros que home.html exceptuando a los cards, carousel y los links del menu del header se ven modificados.
  • Se agrega el .jumbotron para agregar una descripción del clásico 'quienes somos' que tiene toda web.
  • 03 - contact.html
  • Comparte los mismos parámetros que home.html y about.html exceptuando a los cards, carousel y jumbotron, además los links del menú del header se ven modificados.
  • Se implementa el .form-row de bootstrap para crear un formulario de contacto.

Se crea una hoja de estilos personalizada en ./css/style.css donde modifiqué parámetros como:

  • 1 - HEADER
  • Fuente de titulo en header
  • Fuente en los botones
  • BackgroundColor del body
  • BackgroundColor del header
  • Height del header
  • Se aplica display flex al header para ordenar los elementos hijos
  • Se modifica el border-radius del header
  • El carro y el menu están ambos en un mismo div para que el space-between del flex del header no los separe entre si.
  • Se modifica el padding y margin del h5 del header para evitar el desplazamiento
  • 2 - CAROUSEL
  • Se modifica la clase de imagenes d-block del carousel de bootstrap para que tengan un height idéntico y no haya desplazamiento, además se les da un border-radius
  • 3- CARDS
  • Los elementos card de bootstrap están dentro de un section con nombre de clase .cards
  • Se aplica un display flex al section .cards con un flex-flow row para que los elementos hijos estén alineados horizontalmente
  • Se modifica el BackgroundColor de todos los .card, por defecto bootstrap los trae en blanco, se modifican por un #222 gris oscuro
  • Se alinea el texto de los .card
  • A cada imagen contenida dentro de los .card se le aplica un max-height de 180px para que no haya desplazamiento en el tamaño
  • A cada .card-title se le modifica la fuente para que esté acorde a la estética de la web, como el h5 del header, además se le da un sombreado y se le especifica como color #eee
  • A cada .card-text se le aplica un color #eee, esto modifica el color del texto de cada .card
  • 4- BUTTONS
  • A todos los elementos .btn de bootstrap se les modifica la fuente para que estén acorde al estilo de la web. Además se les modifica el color de dicha fuente por un #111
  • 5- FORMS
  • Se modifica el BackgroundColor del form para que sea claro, específicamente se utiliza un color #fff
  • Se le aplica un padding de 10px al form para que sea más agradable a la vista y los bordes no queden tan pegados a los elementos hijos
  • Se aplica un border-radius al form de 05 px para que esté acorde a otros elementos de la web que también tienen border-radius, como por ejemplo el carousel
  • Se centra el texto del form con un text-align: center
  • 6- FOOTER
  • Se centra el texto del footer, además de darle un color blanco al texto, por último se le aplica un padding-top para alejarlo un poco de los demás elementos.
